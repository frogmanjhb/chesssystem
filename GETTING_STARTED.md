# Getting Started with Chess System

## Quick Start Guide for First-Time Users

### 1. Create Your Account
1. Open the Chess System website
2. Click the **"Sign Up"** button in the top-right corner
3. Fill in your details:
   - First Name
   - Last Name
   - Email address (**Important:** Must be your @stpeters.co.za email)
   - Password
4. Click **"Sign Up"** to create your account
5. You'll be automatically logged in!

**Note:** Only St Peter's email addresses (@stpeters.co.za) are allowed to register.

### 2. Create Your First Tournament
1. From the home page, click **"Create New Tournament"**
2. Enter tournament details:
   - **Tournament Name**: Give it a descriptive name (e.g., "Spring Chess Championship")
   - **Max Rounds**: Choose how many rounds to play (3-7 rounds)
   - **Time Control**: Select the time limit per game (e.g., 30+0 means 30 minutes per player)
3. Click **"Create Tournament"**

### 3. Add Students/Players
You have two options:

**Option A: Add Players One at a Time**
1. Enter the student's first name
2. Enter the student's last name
3. Enter their chess rating (use 1200 if you're not sure)
4. Click **"Add Student"**

**Option B: Bulk Import (Faster!)**
1. Click **"Bulk Import"**
2. Type or paste names, one per line:
   ```
   John Doe
   Jane Smith
   Bob Wilson
   ```
3. Click **"Import Students"**

### 4. Generate Your First Round
1. Once you have at least 2 students added, click **"Generate New Round"**
2. The system will automatically create fair pairings using the Swiss system
3. Go to the **"Current Round"** tab to see the matches

### 5. Enter Game Results
1. In the **"Current Round"** tab, you'll see all the pairings
2. For each game, click the appropriate result button:
   - **1-0**: White wins
   - **Â½-Â½**: Draw
   - **0-1**: Black wins
3. Results automatically update the standings!

### 6. View Standings
- Click the **"Standings"** tab to see current rankings
- Players are sorted by score, then by rating
- Check who's leading the tournament!

### 7. Continue Playing
1. After all games in a round are complete, click **"Generate New Round"**
2. The system creates new pairings based on current scores
3. Repeat until you've completed all rounds
4. When finished, click **"End Tournament"**
